# Data Structures

**Definition:** A data structure is a way of organizing data so that it can be accessed and modified efficiently.

## Stack

It is a LIFO (Last In, First Out) data structure. This means that the most recently added element to the stack, will be the first one to be removed.

**Use case:** Undo operations in software, where the last action performed is the first one to be reversed.

## Queue

It is a FIFO (First In, First Out) data structure. This means that the first element added to the queue will be the first one to be removed, similar to a real-life waiting line, where the first person incorporated into the queue, is the first to leave after being attended.

**Use case:** Input buffers (such as keyboard input), where data is processed in the same order it is received (the first keys received are the first ones to be processed and leave the buffer).
